# lurkr-scraper

SM scraper (for personal use only)

## Project Structure

```
lurkr-scraper/
├── main.go                   # Slim Go entry point
├── go.mod
├── backend/
│   ├── cache/
│   │   └── cache.go          # TTL cache module
│   ├── handlers/
│   │   ├── tiktok.go         # TikTok video & comment handlers
│   │   ├── youtube.go        # YouTube channel & video handlers
│   │   └── misc.go           # Health check & image proxy
│   └── utils/
│       └── utils.go          # Shared HTTP, value, and date helpers
├── src/
│   ├── main.ts               # Frontend entry point / DOMContentLoaded bindings
│   └── modules/
│       ├── types.ts          # TypeScript interfaces
│       ├── state.ts          # Shared mutable state
│       ├── theme.ts          # Dark/light theme
│       ├── keys.ts           # API key storage & drawer
│       ├── ui.ts             # Shared UI helpers (format, sanitize, DOM)
│       ├── export.ts         # CSV/XLSX export helpers
│       ├── nav.ts            # Platform & tab switching
│       ├── format.ts         # YouTube format detection (Shorts vs Horizontal)
│       ├── tiktok.ts         # TikTok fetch, render, export
│       ├── wordcloud.ts      # Word cloud generation
│       └── youtube.ts        # YouTube fetch, render, export
├── static/
│   ├── index.html
│   ├── style.css
│   └── app.js                # Compiled & bundled output (generated by build)
├── package.json
└── tsconfig.json
```

## Frontend Build

```bash
# Install dependencies
npm install

# Build (minified, browser-ready bundle → static/app.js)
npm run build

# Watch mode for development
npm run dev

# Type-check only (no emit)
npm run typecheck
```

## Backend Run

```bash
go run main.go
# or
go build -o lurkr && ./lurkr
```

Server starts on port 5000 by default. Set the `PORT` env var to override.
